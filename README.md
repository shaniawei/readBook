2.书架 --阅读历史记录页面(收藏待做)

1.展示书籍列表

2.点击进入阅读页面 

3.每一章节阅读完毕后可跳转至相对应的一套选择题

4.只要点击阅读既可加入书架的历史记录页面

5.记录阅读的进度  实现 接着上一次阅读的位置继续阅读 的功能

6.选择题部分包括单选多选 提交后可显示总得分。
